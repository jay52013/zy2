# 结论
- Matplotlib​

基础绘图库，支持折线图、柱状图等多种图表，通过pyplot接口可快速生成图表，自定义性强
。
​适用场景​：静态图表绘制，适合快速验证数据趋势。
- ​Seaborn​

基于Matplotlib封装，默认样式美观，支持热图、箱线图等统计图表，一行代码生成高质量可视化。
